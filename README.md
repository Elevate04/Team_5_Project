# Team_5_Project
Team 5 Project for Rutgers Bootcamp Course

# Background
For this project, we focused on a specific dataset from the database Kaggle. We want to analyze the trends in the drug-related overdoses in CT. The dataset was titled “Drug Overdose Deaths: Accidental drug related deaths during 2012 - 2018.” This data set provided us with various information of an individual's age, ravce, gender, place of death, cause of death etc. 
Source: https://www.kaggle.com/datasets/ruchi798/drug-overdose-deaths
